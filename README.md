# SimpleTeamColourSprite

A simple sprite node that will adapts its colours based on what team it is set to be.

## Installation

Make sure to download the ChillCube Developer tools: https://github.com/ChillCube/ChillCube-Developer-Tools.git

### Using the Godot Plugin
Enable the **ChillCube Tools** plugin, then paste the repo URL into the Clone tab.

### Linux / CLI
```bash
clone-gd-addon https://github.com/ChillCube/SimpleTeamColourSprite.git
```

## Documentation
https://github.com/ChillCube/SimpleTeamColourSprite/blob/main/DOCUMENTATION.md

## Deinstallation
Use the CC Tools panel in Godot, or run:
```bash
remove-gd-addon SimpleTeamColourSprite
```

## For Developers
```bash
push-all-addons
```
